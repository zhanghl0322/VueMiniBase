# VueMiniBase

>VueMiniBase 是一款个人开箱即用的小前端框架，封装了请求工具类，状态存储，路由等 常用工具类插件
>
#### 主要涉及的插件

>vue-router (路由)\
>vuex (负责用户等相关信息的存储)\
>iconfont (阿里图标库)\
>mock (模拟后端请求)\
>axios (封装了请求工具类)\
>eslint(代码检测)\
>
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:18081 （开发环境地址）
npm run serve

# 打包正式环境，使用的正式环境地址
npm run build

# 自动修复代码格式
npm run lint
```
